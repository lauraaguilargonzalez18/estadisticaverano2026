# Student Performance Dataset

## Description

The **Student Performance Dataset** is designed to examine the factors that influence students' academic performance. It contains **10,000 student records**, where each record includes information about several predictor variables and a performance index.

## Variables

The dataset includes the following variables:

- **Hours Studied:** Total number of hours spent studying by each student.
- **Previous Scores:** Scores obtained by students in previous tests.
- **Extracurricular Activities:** Indicates whether the student participates in extracurricular activities (`Yes` or `No`).
- **Sleep Hours:** Average number of hours the student sleeps per day.
- **Sample Question Papers Practiced:** Number of sample question papers practiced by the student.

## Target Variable

### Performance Index

The **Performance Index** represents the student's overall academic performance. It has been rounded to the nearest integer and ranges from **10 to 100**, where:

- **10** = Lowest academic performance
- **100** = Highest academic performance

Higher values indicate better academic achievement.

## Objective

The dataset aims to provide insights into the relationship between the predictor variables and the **Performance Index**. Researchers and data analysts can use this dataset to explore how the following factors influence student performance:

- Study hours
- Previous academic scores
- Participation in extracurricular activities
- Sleep duration
- Practice with sample question papers

## Notes

> **Note:** This dataset is **synthetic** and was created for illustrative purposes. Therefore, the relationships between the variables and the performance index may not accurately represent real-world scenarios.
